# Identidad visual aplicada — Ruum Ruum Conductor

Este paquete aplica la propuesta visual de Ruum Ruum by MoviliaX a la aplicación de conductor.

## Archivos modificados

- `app/page.tsx`  
  Nueva interfaz visual completa para la app de conductor: Inicio, Viajes, Evidencia, Ganancias y Perfil.

- `app/globals.css`  
  Variables de marca, fondos, tarjetas premium, botones, mockup móvil y animaciones.

- `app/layout.tsx`  
  Metadatos limpios de la aplicación.

- `tailwind.config.ts`  
  Extensión de colores y sombras de identidad Ruum Ruum.

## Paleta aplicada

- Azul Ruum: `#1565FF`
- Azul oscuro: `#0A1F44`
- Verde evidencia: `#00C853`
- Naranja acción: `#FF6D00`
- Fondo claro: `#F8FAFC`
- Texto principal: `#111827`

## Importante

Esta versión está enfocada en aplicar la identidad visual y presentar una interfaz limpia tipo mockup funcional. Si deseas conservar conexión real con Supabase y flujos productivos, integra estos estilos y componentes sobre tus consultas actuales.
